# Hat Monotile Visualizer

Interactive web-based visualization of the aperiodic "hat" monotile discovered by Smith et al. (2023).

## Setup
```bash
pip install flask
python app.py
```

Visit [http://localhost:5000](http://localhost:5000)

## Credits

-   **Original Discovery**: Smith, D., Myers, J. S., Kaplan, C. S., & Goodman-Strauss, C. (2023). An aperiodic monotile. [arXiv:2303.10798](https://arxiv.org/abs/2303.10798)
-   **Single Tile Rendering**: Adapted from interactive tool by Craig S. Kaplan (BSD 3-Clause). [https://cs.uwaterloo.ca/~csk/hat/](https://cs.uwaterloo.ca/~csk/hat/)
